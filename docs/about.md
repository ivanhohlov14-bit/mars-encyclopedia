# О проекте

**Марсианская энциклопедия** — независимый справочный ресурс, созданный по мотивам цикла романов **«Письмо из Красной пыли»** (автор — Mnemis).

## Цель проекта
Систематизировать знания о мире Марса в Эпоху Умирания: историю, географию, культуру, языки, народы, персонажей, события и артефакты.

## Основа энциклопедии
Материалы основаны на глиняных табличках, найденных историком Хевсуром в подземном храме долины Ксанфа.
