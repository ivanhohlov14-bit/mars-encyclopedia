# «Ацидалийское море»

**«Ацидалийское море»** — первая книга цикла «Письмо из Красной пыли», написанная Иваном Хохловым. Издана в 2026 году.

## О книге

Марс умирает. Море, которое веками кормило марсиан, замерзает. Вулканы Фарсиды просыпаются. Время уходит. Одни строят корабли, чтобы улететь к Земле. Другие спускаются в подземные храмы, чтобы записать всё, что помнят.

## Персонажи

- **Хевсур** — историк, который ищет правду в древних табличках.
- **Талин** — астроном, который смотрит на звёзды.
- **Элла** — биолог, сестра Талина.
- **Аратан III** — последний король Эдема.
