# «Ацидалийское море»

**«Ацидалийское море»** — первая книга цикла «Письмо из Красной пыли», написанная Иваном Хохловым. Издана в 2026 году.

## О книге

Марс умирает. Море замерзает. Вулканы просыпаются. Герои строят корабли, чтобы улететь к Земле.

## Персонажи

- **Хевсур** — историк.
- **Талин** — астроном.
- **Элла** — биолог.
- **Аратан III** — последний король.
