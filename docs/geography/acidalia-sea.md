# Ацидалийское море

**Ацидалийское море** — крупнейший водоём Марса в Эпоху Умирания.

## Характеристики

| Параметр | Значение |
|----------|----------|
| **Площадь** | 1,5 млн км² |
| **Солёность** | ~35 ‰ |
| **Средняя глубина** | 150–200 м |

## История

С 2650 года уровень моря начал падать. К 2735 году море полностью замёрзло.
