# Свиток Ж. Языки и народы

## Язык Marzān
Основной язык марсиан. Имеет три диалекта:
- Южный (Эллада)
- Западный (Утопия)
- Центральный (Окхасен)

## Народы
- **Тёмно-синие** — южные моря.
- **Светло-синие** — западные равнины.
- **Смешанный тип** — центральные области.
